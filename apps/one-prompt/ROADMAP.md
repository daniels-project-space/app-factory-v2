# OnePrompt — Product Roadmap

> **One Thought. One Day. Forever.**
> Minimal micro-journaling with time capsules, local weekly reflections, photo journaling, mood tracking, and philosopher guides.

## Current reflection behavior

- [x] Premium-gated weekly reflection screen with summary, theme, mood chart, encouragement, and refresh.
- [x] Deterministic on-device rules for text-only, photo-only, mixed, and empty weeks.
- [x] Curated philosopher-guide and journaling-goal variants.
- [x] Versioned local cache, invalidated when entries, goals, guide, or engine rules change.
- [x] Photo-count insights that do not inspect image content.
- [x] No server reflection path or reflection rate-limit state.

## Remaining release checks

- [ ] Accessibility audit: VoiceOver, TalkBack, Reduce Motion, and contrast.
- [ ] Test offline mode on physical iOS and Android devices.
- [ ] Test PDF export across themes with varied entry data.
- [ ] Finalize App Store screenshots and listing copy.
- [ ] Build production releases after store review.
